Project Title:
CPU load dynamic monitoring

Summary:
In this project, I had the task of creating a Java application that would monitor the processor load in real time, until the application stopped. In order to be able to make the graphics in real time I used an external library: JFreechart.


Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.


Prerequisites and Installing:
To run my sample you need to install the following ->
-> Java version 8 : https://java.com/en/download/help/download_options.xml
->Install what java ide you like
->About JFreechart: http://www.jfree.org/jfreechart/
